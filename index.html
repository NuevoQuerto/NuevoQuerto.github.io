        a. Metode Insertion

Insertion Sort adalah suatu metode urut yang melakukan pengurutan dengan cara menyisipkan data yang belum urut ke dalam bagian data yang telah diurutkan. Konsep seperti ini biasa dilakukan pada permainan kartu.

Algoritma:

procedure insertionSort(input/output array : Array of integer, input ukuranArray : integer)

Deklarasi:

i : integer

j : integer

x : integer

Deskripsi Algoritma:

for i ← 1 to ukuranArray do

        x ← array[i]

        j ← i - 1

        ketemu ← false

        while((j  ≥ 0 ) and (not ketemu)) do

                if x < array[j] then

                        array[j + 1] ← array[j]

                        j ← j - 1

                else

                        ketemu ← true

                endif

        endwhile

        L[j + 1] ← x

endfor

        b. Metode Selection

Gagasan metode ini adalah memilih elemen maksimum/minimum dari array, lalu menempatkan elemen maksimum/minimum itu pada ujung (awal atau akhir) array. Selanjutnya elemen terujung tersebut diisolasi dan tidak disertakan pada proses selanjutnya. Proses yang sama diulang untuk elemen larik yang tersisa. Pemilihan nilai maksimum / minimum dilakukan pada setiap pass.

Ditinjau dari pemilihan elemen maksimum/minimum, algoritma pengurutan seleksi dibagi menjadi dua:

1. Algoritma pengurutan seleksi-maksimum, yaitu memilih elemen maksimum sebagai basis pengurutan.

2. Algoritma pengurutan seleksi-minimum, yaitu memilih elemen minimum sebagai basis pengurutan.

Algoritma Selection Maksimum:

procedure selectionMax(input/output array: Array of integer, input ukuranArray : integer)

Deklarasi:

i : integer (pencacah pass)

j : integer (pencacah untuk mencari nilai maksimum)

imaks : integer (indeks yang berisi nilai maksimum sementara)

tmp : integer (variabel bantu untuk pertukaran)

Deksripsi Algoritma:

for i ← ukuranArray downto 1 do

        imaks ← 0

        for j ← 1 to i do

                if array[j] > array[imaks] then

                        imaks ← j

                endif

        endfor

        tmp ← array[i]

        array[i] ← array[imaks]

        array[imaks] ← tmp

endfor

        c. Metode Bubble

Bubble Sort merupakan metode tersederhana untuk melakukan pengurutan data, tetapi memiliki kinerja terburuk untuk data yang besar. Pengurutan dilakukan dengan membandingkan sebuah bilangan dengan seluruh bilangan yang terletak sesudah bilangan tersebut. Penukaran dilakukan kalau suatu kriteria dipenuhi.

Algoritma:

procedure bubbleSort(input/output array : Array of integer, input ukuranArray : integer)

Deklarasi:

i : integer (pencacah untuk jumlah langkah)

k : integer (pencacah untuk pengapungan pada setiap langkah)

tmp : integer (variabel bantu untuk pertukaran)

        d. Metode Shell

Metode ini merupakan perbaikan terhadap metode Insertion Sort. Insertion Sort mempunyai keunggulan bila data yang diurutkan sudah hampir urut. Tetapi untuk data yang urutannya terbalik atau banyak data yang berada didaerah berlawanan maka akan banyak sekali proses penggeseran.

Untuk mengurangi pergeseran terlalu jauh maka larik diurutkan setiap k elemen dengan menggunakan metode Insertion Sort( k dinamakan step atau increment ), misak k=5. Selanjutnya digunakan nilai step yang lebih kecil, misal k=3, lalu diurut setiap 3 elemen.  Begitu seterusnya sampai nilai k=1.

Algoritma:

procedure shellSort(input/output array : Array of integer, input ukuranArray : integer)

Deklarasi:

gap : integer

i : integer

j : integer

temp : integer

Deskripsi Algoritma:

gap ← ukuranArray / 2

while gap > 0 do

        i ← gap

        while i < ukuranArray do

                temp ← array[i]

                j ← i

                while j ≥ gap and array[j - gap] > temp do

                        array[j] ← array[j - gap]

                        j ← j - gap

                endwhile

                array[j] = temp

                i ← i + 1

        endwhile

        gap ← gap / 2

endwhile

e. Metode Quick

Quick Sort adalah metode pengurutan data yang ditemukan pertama kali oleh C. A. R. Hoare pada 1962. Metode ini menggunakan strategi “pecah-belah” dengan mekanisme seperti berikut. Larik L[p..r] (dengan indeks terkecil adalah p dan indeks terbesar, yaitu r) disusun ulang (dipartisi) menjadi dua larik A[p..q] dan A[q+1..r] sehingga setiap elemen dalam A[p..q] selalu bernilai lebih kecil daripada setiap nilai elemen pada A[q+1..r]. Selanjutnya. kedua larik tersebut diurutkan secara rekursif. Dengan sendirinya, kombinasi kedua larik tersebut membentuk larik dengan data yang telah urut.

Algoritma:

procedure quickSort(input/output array: Array of integer, input kiri : integer, input kanan : integer)

Deklarasi:

tmp : integer

i : integer

j : integer

pivot : integer

Deskripsi Algoritma:

i ← kiri

j ← kanan

pivot ← array[(kiri + kanan) / 2]

while i ≤ j do

        while array[i] < pivot do

                i ← i + 1

        endwhile

        while array[j] > pivot do

                j ← j - 1

        endwhile

        

if i ≤ j then

        tmp ← array[i]

        array[i] ← array[j]

        array[j] ← tmp

        i ← i + 1

        j ← j - 1

endif

        endwhile

        if kiri < j then

                quickSort(array, kiri, j)

        endif

        if i < kanan then

                quickSort(array, i, kanan)

        endif

f. Metode Merge

Merge Sort merupakan algoritme pengurutan dalam ilmu komputer yang dirancang untuk memenuhi kebutuhan pengurutan atas suatu rangkaian data yang tidak memungkinkan untuk ditampung dalam memori komputer karena jumlahnya yang terlalu besar. Algoritme ini ditemukan oleh John von Neumann pada tahun 1945.

Prinsip utama yang diimplementasikan pada algoritme Merge Sort (urut gabung) seringkali disebut sebagai pecah-belah dan taklukkan (divide and conquer). Cara kerja algoritme Merge Sort (urut gabung) adalah membagi larik data yang diberikan menjadi dua bagian yang lebih kecil. Kedua larik yang baru tersebut kemudian akan diurutkan secara terpisah. Setelah kedua buah list tersusun, maka akan dibentuk larik baru sebagai hasil penggabungan dari dua buah larik sebelumnya.

Menurut keefektifannya, alogaritma ini bekerja dengan tingkat keefektifan O(nlog(n)).

Algoritma:

procedure merging(input/output a : Array of integer, input/output b : Array of integer, input low : integer, input mid : integer, input high : integer)

Deklarasi:

l1 : integer

l2 : integer

i : integer

        Deskripsi Algoritma:

        l1 ← low

        l2 ← mid

        i ← low

        while l1 ≤ mid and l2 ≤ high do

                if a[l1] ≤ a[l2] then

                        b[i] = a[l1 ← l1 + 1]

                else

                        b[i] = a[l2 ← l2 + 1]

                endif

                i ← i + 1

        endwhile

        while l1 ≤ mid do

                b[i + 1] ← a[l1 ← l1 + 1]

        endwhile

        while l2 ≤ high do

                b[i + 1] ← a[l2 ← l2 + 1]

        endwhile

        for i ← low to high do

                a[i] ← b[i]

        endfor

        Algoritma:

function sort(input/output a : Array of integer, input/output b : Array of integer, input low : integer, input high : integer)

        Deklarasi:

        mid : integer

        Deskripsi Algoritma:

        if low < high then

                mid ← (low + high) / 2

                sort(a, b, low, mid)

                sort(a, b, mid + 1, high)

                merging(a, b, low, mid, high)

        else

                return

        endif

- Searching

a. Metode Linear/Sequential

Pencarian sekuensial (atau disebut juga pencarian linear) merupakan model pencarian yang paling sederhana yang dilakukan terhadap suatu kumpulan data. Secara konsep pencarian sekuensial, membandingkan setiap elemen larik satu persatu secara beruntun, mulai dari elemen pertama sampai dengan elemen yang dicari ditemukan atau seluruh elemen sudah ditemukan.

Algoritma:

function sequentialSearch(input array : Array of integer, input ukuranArray : integer, input cari : integer)

Deklarasi:

i : integer

Deskripsi Algoritma:

i ← 0

while i < ukuranArray do

        if array[i] = cari then

                return array[i]

        endif

        i ← i + 1

endwhile

b. Metode Binary

Metode ini memerlukan data yang sudah terurut. Pencarian Biner dilakukan dengan membagi larik menjadi dua bagian dengan jumlah yang sama atau berbeda 1 jika jumlah data semula ganjil. Data yang dicari kemudian dibandingkan dengan data terakhir pada bagian utama. Dalam hal ini, ada tiga kemungkinan yang terjadi.

1. Data yang dicari sama dengan elemen terakhir pada bagian pertama dalam larik. Jika kondisi ini terpenuhi, data yang dicari berarti ditemukan.

2. Data yang dicari bernilai kurang dari nilai elemen terakhir pada bagian pertama dalam larik. Pada keadaan ini, pencarian diteruskan pada bagian pertama.

3. Data yang dicari bernilai lebih dari elemen terakhir pada bagian pertama dalam larik. Pada keadaan ini, pencarian diteruskan pada bagian kedua.

Algoritma:

function binarySearch(input array : Array of integer, input ukuranArray : integer, input cari : integer)

Deklarasi:

min : integer

max : integer

tebakan : integer

Deskripsi Algoritma:

min ← 0

max ← ukuranArray - 1

while max ≥ min do

        tebakan ← (min + max) / 2

        if array[tebakan] = cari then

                return tebakan

        else if array[tebakan] < cari then

                min ← tebakan + 1

        else

                max ← tebakan - 1

        endif

endwhile
